# CoreForge supplement images — catalog

Reference codes (`CFG-xxx`) are stable IDs for commerce, CMS, and filenames. **Prompt text is intentionally omitted** from this file.

## Summary

| Status | Count |
|--------|------:|
| Renamed on disk | 33 |
| Alternate flavor / variant (same list price, second file) | 1 (`CFG-023B`) |
| **No matching image in folder** | 2 (`CFG-019`, `CFG-025`) |

**Duplicates removed:** four files were byte-identical copies of another asset but saved under the wrong product name (`109` Vegan copy of Collagen, `149` Strength Duo copy of Protein + Greens, `89` Pre-Workout Energy copy of Advanced Pre-Workout, `399` Complete System copy of Recovery Protein). Those extras were deleted; the canonical files below were kept.

**Vision / label checks:** filenames were matched to your price list using label text and packaging type. A few labels differ slightly from the marketing line (weights or flavor names); notes call that out.

## Label / asset notes (no prompts)

- **CFG-016:** Pack reads “Collagen Protein” with a smaller net weight on-pack than some listings; treat as the Collagen SKU art.
- **CFG-017:** Label shows **Protein + Greens**, ~500g / 20 servings on-pack — your shop copy may say 2kg; align PDP copy with the real pack or re-render.
- **CFG-020:** Kraft-style vegan protein; label shows **~1 kg** net weight while the line item may say “blend”; align copy if needed.
- **CFG-023 / CFG-023B:** Two legitimate renders for **Casein 2kg** (vanilla vs chocolate). Use **CFG-023** as default PDP hero unless the variant is chocolate.
- **CFG-029:** Label shows **~2 lb / 907g**; listing says 1kg — minor mismatch; update one side for consistency.
- **CFG-019:** The file that was named for this SKU was an **exact duplicate** of **CFG-015** (Advanced Pre-Workout 60 Serves). There is **no separate yellow-accent “Pre-Workout Energy”** image in this folder — add a new render if you need that SKU.
- **CFG-025:** No bundle in the folder matches **Platinum Stack** (premium multi-product, gold-accent bundle). Add a new asset for this SKU.

## Script

From the repo root:

```bash
python3 other/supp-images_rename.py --dry-run
python3 other/supp-images_rename.py
```

The script deletes the four misnamed duplicate files, then renames sources to the `CFG-xxx_{price} — {Title}.png` pattern above.
